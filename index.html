Census Data Connector

About

Simple interactive webmap that allows users to select the level geography they'd like data for (county, municipality, tract, block group, block), then click on their area of interest to access the Census Bureau's data profile (example here). The Census Bureau data profile contains the most current decennial census counts, as well as most recent 5-year ACS data, depending on the geography selected. From this page, the user has the ability to download whatever data they'd like, directly from data.census.gov.

Motivation

DVRPC's Data Coordination & Analysis work program area contains the following tasks: “Participate in and promote Census Geography update processes and products” and “Disseminate Census-related data through DVRPC internal and external resources.” I get contacted regularly from planning partners (county planners as well as municipalities) asking if DVRPC can provide them with Census data. They are finding data.census.gov too difficult to use. The Census data profiles get users quick access to ACS and decennial data, organized by theme. Additionally, this solution will eliminate the need for DVRPC to store and serve out ACS data on our Data Navigator. The Data Navigator does not give end-users margin of error data, nor does it provide metadata about the datasets, which is problematic and something DVRPC should discontinue.

Limitations

Users will not have the (easy) ability to download data for more than one geographic unit at a time.
